# 🌍 Air Quality Index (AQI) Analyzer

## 📌 Project Overview

The **Air Quality Index (AQI) Analyzer** is a web-based application that monitors and analyzes air pollution levels across different regions using air quality datasets. The system processes pollutant data such as **PM2.5, PM10, NO₂, SO₂, CO, and O₃** to evaluate air quality and provide meaningful insights through visualizations.

This project helps users understand pollution trends, compare air quality between different regions, and identify highly polluted areas.

---

## 🎯 Objectives

- Monitor air pollution across different regions.
- Analyze air quality using pollutant data.
- Compare pollution levels between cities or regions.
- Display AQI information through an interactive dashboard.
- Provide easy-to-understand visualizations for users.

---

## ✨ Features

- 📊 Analyze air quality data from multiple regions.
- 🌍 Compare pollution levels across cities.
- 📈 Visualize AQI trends using charts.
- 🔍 Filter air quality data by region.
- 📋 Display pollutant values such as PM2.5, PM10, NO₂, SO₂, CO, and O₃.
- 💻 User-friendly web interface.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Air-Quality-Index-Analyzer.git
```

### Navigate to Project Folder

```bash
cd Air-Quality-Index-Analyzer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
cd backend
python app.py
```
